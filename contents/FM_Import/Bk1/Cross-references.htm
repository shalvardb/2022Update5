Cross-references

There are several predefined cross-reference formats. chief among them are:

•​Heading

•​Heading & Page

•​ChapterNum

•​ChapterNum & Name

•​ChapterNum & Name & Page

The first results in a cross-reference that looks like this: For more information, see the “Paragraph formats” section.

The second looks like this: For more information, see the “Paragraph formats” on page 1 section.

The chapter cross-reference formats look like the following examples:

•​For more info, see Chapter 1. (Select the Chapter.number format in the Cross-Ref dialog box.)

•​For more info, see Chapter 1, “A Sample Chapter13 4”. (Use the Chapter.name format in the Cross-Ref dialog box.)

•​For more info, see 6C:\Users\shalvar\AppData\Roaming\Adobe\FrameMaker\17. (Use the Chapter.name format in the Cross-Ref dialog box.)
