Creating books

In addition to this template for chapters or appendixes, there are templates for a table of contents and an index. All the templates are gathered into a sample book file.

Here’s a summary of how you would use these templates to create your own book based on this design.

To create a book from scratch:

1​Open the book template and rename it to an appropriate name. This changes the names of the generated files automatically.

2​Remove the chapter file from the book file and insert your own. Be sure to set up the first chapter file so that it restarts numbering (choose File > Set Up File).

3​Save the book file.

4​Create the chapter files, using the chapter file template as the template for the new files.

5​Return to the book file and generate and update it (choose File > Generate/Update). This process creates the TOC and index for the first time.

6​Import formats from the TOC template to the newly created TOC file (File > Import > Formats).

7​Import formats from the index template to the newly created index file.

8​Generate and update again (File > Generate/Update).

To create a book from legacy documents:

1​Import all formats from the chapter template to the existing chapter files.

2​Retag the files as necessary (this might be a labor-intensive chore).

3​Follow the steps in the preceding procedure.
